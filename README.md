# GON-Detector

## 项目简介：

这是我的大创作品。我们使用了三个卷积神经网络来对眼底图片进行诊断，判断患青光眼的概率。

## 项目主要参与人员：

* ZT
* SMQ
* XZX

## 项目主要数据集：

* REFUGE青光眼竞赛

## 项目实现：

#### 三个模型（具体见代码）：

* Inception-Resnet-V2模型
* U-net模型
* M-net模型

#### 实验环境：

* python 3.7+
* pytorch 1.6.0
* tensorflow 2.1.0
* keras 2.3.1
* paddle-paddle 1.8.0
